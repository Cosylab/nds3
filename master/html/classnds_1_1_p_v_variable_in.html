nds::PVVariableIn< T > Class Template Reference

Detailed Description

template<typename T>
class nds::PVVariableIn< T >

An input PV object that also stores the value so it does not need to delegate the read function.

The device support can use PVVariableIn::setValue() to modify the PV's value; the read() method will be used to read the value on the control system side.

Warning
for all the vector data types and for std::string remember to call setMaxElements() to specify the maximum size of the vector or string.
Template Parameters
Tthe PV data type. The following data types are supported:
  • std::int32_t
  • std::double
  • std::vector<std::int8_t>
  • std::vector<std::uint8_t>
  • std::vector<std::int32_t>
  • std::vector<double>
  • std::string
